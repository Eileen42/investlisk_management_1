핵심 투자 원칙 3가지

🌊
원칙 1: 시장을 이기려 하지 마세요
추세의 흐름을 읽고 분산투자로 리스크를 줄이는 것이 최고의 전략이에요. 시장 흐름은 개인이 이길 수 없어요. 추세를 탈 줄 아는 개미가 됩시다.
📅
원칙 2: 꾸준히, 정기적으로
매달 같은 날 같은 금액을 투자하세요. 가격이 쌀 때 더 많이 사지게 돼요. 현금흐름 통제소에서 자동 일정을 관리해요.
🛑
원칙 3: 하락장엔 멈추세요
월봉 10이평선 아래로 내려가면 매수를 멈추세요. 추세 통제탑이 자동으로 판단해줘요.

처음 시작하기 (4단계)

1
종목 등록
→ 마스터 데이터
마스터 데이터 탭에서 보유 중인 주식·ETF·코인을 등록해요.
[거래 추가] → [종목추가]를 선택해 첫 종목을 등록하세요.
👆 이 카드를 클릭하면 바로 이동해요
2
추세 확인
→ 추세 통제탑
추세 통제탑 탭에서 [추세 자동 조회] 클릭.
각 종목의 월봉 10이평선이 자동 계산되고 1~5단계 국면이 표시돼요.
👆 이 카드를 클릭하면 바로 이동해요
3
투자 계획 설정
→ 현금흐름 통제소
현금흐름 통제소 탭에서 각 종목별 자동투자 주기와 1회 투자금액을 설정해요.
매달 얼마씩 넣을지 계획하고 실행 여부를 체크해요.
👆 이 카드를 클릭하면 바로 이동해요
4
알림 설정
→ 알림 설정
헤더의 🔔 버튼을 눌러 텔레그램 봇 또는 브라우저 알림을 설정해요.
금요일/월말에 10이평선 이탈 시 자동으로 매도 신호를 알려드려요.
👆 이 카드를 클릭하면 알림 설정 창이 열려요

📅 월간 루틴 (매월 말일)

1
현재가 자동 조회 → 마스터 데이터
마스터 데이터 탭에서 [현재가 자동 조회] 클릭 → 전 종목 현재가 일괄 업데이트
2
추세 자동 조회 → 추세 통제탑
추세 통제탑 탭에서 [추세 자동 조회] 클릭 → 월봉 10이평선 + 국면 자동 판단
3
행동 지침 확인 → 추세 통제탑
🔥 1단계(찐상승장)이면 계속 모으기 / ❄️ 4단계(빙하기)이면 전량 매도 검토
4
월간 결산 확인 → 월간 결산표
이번 달 투자 원금, 평가금액, 수익률, 잡초 종목을 확인해요
5
잡초 점검 → 마스터 데이터
수익률 -20% 이하 종목(잡초)은 정리 여부 검토

Q. 10이평선은 어떻게 계산되나요?→ 추세 통제탑
최근 10개월 월봉 종가의 평균값이에요. [추세 자동 조회]를 누르면 Yahoo Finance에서 자동 계산해요.
Q. 잡초는 무조건 팔아야 하나요?
아니에요! 잡초 판별은 참고용이에요. 기업 전망이 좋다면 보유할 수 있어요. 다만 -20% 이상이면 리스크 점검이 필요해요.
Q. 데이터는 어디에 저장되나요?
내 Google Drive의 주린이 떡락방지 DB 스프레드시트에 저장돼요. 서버에는 아무것도 저장되지 않아요.

일별 손익 자동 기록 설정

Google Apps Script를 설정하면 앱 접속 없이 매일 자동으로 기록돼요.

📋 설정 방법 (3단계)
  1. 내 구글 시트 열기 → 상단 메뉴 확장프로그램 → Apps Script
  2. 아래 코드 전체 복사 → 붙여넣기 → 💾 저장 (Ctrl+S)
  3. 트리거 추가: 시계 아이콘(⏰) → + 트리거 → saveMockSnapshot → 시간 기반 → 매일 → 오후 6시~7시
📄 복사할 코드 
// ═══ 모의투자 일별 스냅샷 자동 기록 ═══
// 구글 시트 > 확장프로그램 > Apps Script 에 붙여넣기
// 트리거: saveMockSnapshot > 시간 기반 > 매일 > 오후 6시~7시

var MOCK_SHEET_NAME      = "모의투자";
var MOCK_SNAP_SHEET_NAME = "모의투자_스냅샷";

function saveMockSnapshot() {
  var ss       = SpreadsheetApp.getActiveSpreadsheet();
  var mockSheet = ss.getSheetByName(MOCK_SHEET_NAME);
  var snapSheet = ss.getSheetByName(MOCK_SNAP_SHEET_NAME);
  if (!mockSheet || !snapSheet) {
    Logger.log("시트를 찾을 수 없어요: " + MOCK_SHEET_NAME + " / " + MOCK_SNAP_SHEET_NAME);
    return;
  }

  // 자본금 (모의투자 시트 J1)
  var capitalCell = mockSheet.getRange("J1").getValue();
  var capital = capitalCell ? parseFloat(capitalCell) : 10000000;

  // 거래 내역 읽기 (A2:H)
  var trades = mockSheet.getRange("A2:H" + mockSheet.getLastRow()).getValues()
    .filter(function(r){ return r[0]; })
    .map(function(r){
      return { date: r[0], name: r[1], ticker: r[2], type: r[3],
               qty: parseFloat(r[4])||0, price: parseFloat(r[5])||0, fee: parseFloat(r[6])||0 };
    });

  // 보유 종목 계산
  var holdings = {};
  var cash = capital;
  trades.forEach(function(t) {
    var key = t.ticker || t.name;
    if (!holdings[key]) holdings[key] = { ticker: t.ticker, name: t.name, qty: 0, avgPrice: 0, totalCost: 0 };
    if (t.type === "매수" || t.type === "종목추가") {
      var newCost = holdings[key].totalCost + t.qty * t.price;
      holdings[key].qty += t.qty;
      holdings[key].avgPrice = holdings[key].qty > 0 ? newCost / holdings[key].qty : 0;
      holdings[key].totalCost = newCost;
      cash -= t.qty * t.price + (t.fee || 0);
    } else if (t.type === "매도") {
      holdings[key].qty = Math.max(0, holdings[key].qty - t.qty);
      holdings[key].totalCost = holdings[key].avgPrice * holdings[key].qty;
      cash += t.qty * t.price - (t.fee || 0);
    }
  });

  // 보유 종목 현재가 조회 후 평가액 계산
  var evalAmt = 0;
  Object.values(holdings).forEach(function(h) {
    if (h.qty <= 0) return;
    var price = fetchPrice(h.ticker);
    evalAmt += h.qty * (price > 0 ? price : h.avgPrice);
  });

  var totalValue = Math.round(cash + evalAmt);
  var today = Utilities.formatDate(new Date(), "Asia/Seoul", "yyyy-MM-dd");

  // 오늘 날짜 이미 있으면 업데이트, 없으면 추가
  var existing = snapSheet.getDataRange().getValues();
  var found = -1;
  for (var i = 1; i < existing.length; i++) {
    var rowDate = existing[i][0] instanceof Date
      ? Utilities.formatDate(existing[i][0], "Asia/Seoul", "yyyy-MM-dd")
      : String(existing[i][0]);
    if (rowDate === today) { found = i + 1; break; }
  }
  if (found > 0) {
    snapSheet.getRange(found, 1, 1, 2).setValues([[today, totalValue]]);
  } else {
    snapSheet.appendRow([today, totalValue]);
  }
  Logger.log("모의투자 스냅샷 저장: " + today + " | 총평가액: ₩" + totalValue.toLocaleString());
}

function fetchPrice(ticker) {
  if (!ticker) return 0;
  try {
    // 업비트 코인 (KRW-xxx 형식)
    if (ticker.indexOf("KRW-") === 0) {
      var res = UrlFetchApp.fetch(
        "https://api.upbit.com/v1/ticker?markets=" + encodeURIComponent(ticker),
        { muteHttpExceptions: true }
      );
      var data = JSON.parse(res.getContentText());
      return data && data[0] ? parseFloat(data[0].trade_price) : 0;
    }
    // 주식 (Yahoo Finance)
    var url = "https://query1.finance.yahoo.com/v8/finance/chart/" + encodeURIComponent(ticker)
              + "?interval=1d&range=1d";
    var resp = UrlFetchApp.fetch(url, { muteHttpExceptions: true });
    var json = JSON.parse(resp.getContentText());
    return parseFloat(json.chart.result[0].meta.regularMarketPrice) || 0;
  } catch(e) {
    Logger.log("가격 조회 실패: " + ticker + " / " + e);
    return 0;
  }
}
